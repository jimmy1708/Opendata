* IDENTIFIANT : RFF_LR_LGV

* TITRE : Tracé contournement LGV Nîmes-Montpellier

* DESCRIPTION : Cette donnée renseigne le tracé de la Ligne à Grande Vitesse (LGV) entre Nîmes et Montpellier. Il s'agit du contournement Nîmes-Montpellier sous l'autorité de Réseau Ferré de France (RFF) avec comme constructeur partenaire Oc'via. 

Ce jeu de données contient 3 données distinctes à savoir : le tracé d'avant projet d’avril 2013 du contournement Nîmes-Montpellier (linéaire, Trace_avant_projet_CNM_Avril2013), les principaux points de travaux (ponctuel, Points_importants_CNM), comme les ouvrages d'art de franchissement et enfin le tracé de déclaration d’utilité publique (DUP) du contournement Nîmes-Montpellier (linéaire, Bande_DUP_CNM). 

Les données sont disponibles dans un fichier compressé en 3 formats (shape, kml, geojson) pour chacune des données. 

Vous trouverez des informations complémentaires, comme des représentations cartographiques, sur le site de Réseau Ferré de France (RFF) http://contournement-nimes-montpellier.com/, mais des détails sur les travaux via le site Oc'via, http://www.ocvia.fr/.
